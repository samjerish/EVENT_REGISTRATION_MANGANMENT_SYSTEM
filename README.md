**Event Registration Management System**

Overview : Developed a web-based event registration platform to simplify large-scale participant data management. 
The system collects registration data through a clean interface, stores it in Firebase in real time, and automates Excel export, eliminating manual spreadsheet handling.

Features:
- Event registration form
- Real-time Firebase database integration
- Automated Excel export
- Efficient and organized data management

Technologies Used:
- HTML
- CSS
- JavaScript
- Firebase
  
Impact:Reduced manual workload, improved data accuracy, and enabled smoother event management during large-scale events.
